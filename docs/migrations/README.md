# Database migrations

SQL migrations for the Caicos Supabase project live in a **single** folder at the repository root:

**[`/supabase/migrations/`](../../supabase/migrations/)**

Run them with the Supabase CLI from the repo root (or your linked project), e.g. `supabase db push` / `supabase migration up`.

This `docs/migrations/` path is kept only as a pointer so older notes and links can be updated gradually.
