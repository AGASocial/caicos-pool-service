## 1. Brand Overview

### Brand Name
**PoolDash**

### Tagline
"Service Management, Synchronized"

### Mission
Empower pool service companies to coordinate technicians, manage operations, and deliver exceptional service through intelligent scheduling and real-time visibility.

### Brand Essence
Modern, reliable, efficient, professional

## 2. Logo

### Primary Logo
- **Style:** Minimalist, modern
- **Elements:**
  - Stylized water droplet or dashboard-inspired wave
  - "PoolDash" wordmark in modern sans-serif
  - Square format for app icon compatibility
  - Horizontal lockup for web/marketing

### Logo Variations
1. **Full Logo:** PoolDash name + icon
2. **Icon Only:** Icon mark alone (for app home screen)
3. **Horizontal Lockup:** Icon left, text right (for headers)
4. **Stacked:** Icon above text (for narrow spaces)

### Logo Files Needed
- `logo-primary.png` (transparent, 1000px)
- `logo-icon.png` (square, 512px)
- `logo-horizontal.png` (transparent, 2000px wide)
- `logo-favicon.ico` (16px, 32px, 64px versions)

### Clear Space & Sizing
- Minimum size: 64px (icon only)
- Clear space: Equal to the height of the icon
- Never rotate, skew, or distort the logo

### Color Usage Guidelines

**Web Applications:**
- Primary actions: Teal (#0891B2)
- Backgrounds: White or Slate 50
- Text: Slate 900 (headings), Slate 600 (body)
- Borders: Slate 200
- Success indicators: Green (#10B981)
- Warnings: Amber (#F59E0B)
- Errors: Red (#EF4444)

**Mobile Applications:**
- Same as web
- Ensure sufficient contrast (WCAG AA minimum)

**Avoid:**
- Using Teal and Green together (both signify different states)
- Using color alone to convey information (use icons + text)
- Low contrast combinations (test with WebAIM contrast checker)

## 4. Typography

### Font Stack

**Primary Font: Inter**
- Weight: 400 (Regular), 500 (Medium), 600 (Semibold), 700 (Bold)
- Use for: All UI text, body copy, buttons
- Fallback: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif

**Secondary Font: JetBrains Mono**
- Weight: 400 (Regular), 600 (Semibold)
- Use for: Code, technical data, monospace elements
- Fallback: "Courier New", monospace

### Type Scales

**Web/Admin Portal**
```
H1 (Display):    32px, 700, line-height 1.2
H2 (Large):      28px, 600, line-height 1.3
H3 (Medium):     24px, 600, line-height 1.4
H4 (Small):      20px, 600, line-height 1.4
H5 (Tiny):       16px, 600, line-height 1.5

Body (Large):    16px, 400, line-height 1.6
Body (Regular):  14px, 400, line-height 1.6
Body (Small):    12px, 400, line-height 1.5

Button:          14px, 600, uppercase letters
Label:           12px, 500, uppercase letters
```

**Mobile App**
```
H1:              28px, 700, line-height 1.2
H2:              24px, 600, line-height 1.3
H3:              20px, 600, line-height 1.4

Body (Large):    16px, 400, line-height 1.6
Body (Regular):  14px, 400, line-height 1.6
Body (Small):    12px, 400, line-height 1.5

Button:          16px, 600
Label:           12px, 500
```

### Typography Rules

- **Headings:** Use 600-700 weight, limited line length for readability
- **Body text:** Use 400 weight, 1.6 line-height for comfortable reading
- **Links:** Teal (#0891B2), underlined on hover
- **Emphasis:** Use 600 weight, not italics
- **Code:** Use JetBrains Mono in light gray background
- **All caps:** Reserved for labels and buttons only

## 5. Tone of Voice

### Brand Personality Traits
- **Professional** — Trustworthy, competent
- **Modern** — Contemporary, forward-thinking
- **Helpful** — Clear, supportive, solution-oriented
- **Efficient** — Concise, direct, action-focused
- **Friendly** — Approachable, not corporate-stiff

### Writing Principles

**Do:**
- ✅ Use active voice ("Schedule jobs" not "Jobs can be scheduled")
- ✅ Be direct and specific ("Complete your profile in 2 minutes")
- ✅ Use conversational language ("You've got this!" not "Congratulations on your achievement")
- ✅ Focus on benefits ("Save 2 hours per week" not "Automated scheduling system")
- ✅ Use second person ("You'll see real-time updates")

**Don't:**
- ❌ Use jargon or technical terms without explanation
- ❌ Use negative framing ("Don't forget" → "Remember to")
- ❌ Be overly casual or use slang
- ❌ Use exclamation marks excessively
- ❌ Use passive voice

### Example Copy

**Friendly:**
"Track your team's progress in real-time. See who's completed jobs, who's up next, and manage any issues instantly."

**Not:**
"Real-time job status visualization system with technician assignment management capabilities."

## 6. Visual Style

### Design System Approach
- **Framework:** Tailwind CSS (web), NativeWind (mobile)
- **Component Library:** Shadcn/ui (web)
- **Icons:** Ionicons (consistent across platforms)

### Visual Characteristics

**Spacing:**
- Base unit: 8px
- Common: 8px, 12px, 16px, 20px, 24px, 32px, 40px

**Border Radius:**
- Buttons: 8-12px
- Cards: 12-16px
- Modals: 16px

**Shadows:**
- Light: `0 1px 2px rgba(0,0,0,0.05)`
- Medium: `0 4px 6px rgba(0,0,0,0.1)`
- Large: `0 10px 15px rgba(0,0,0,0.15)`

**Animations:**
- Transitions: 200ms ease-in-out
- Duration: Keep animations snappy (< 300ms)
- Easing: Use ease-in-out for most interactions

### Photography & Imagery

**Style:**
- Professional, clean, modern
- Real-world pool service scenarios
- Technicians in action (with permission)
- Equipment close-ups

**Avoid:**
- Stock photos that look fake
- Overly saturated colors
- Cluttered backgrounds
- Irrelevant imagery

### Icons

**Icon Set:** Ionicons
- Consistent size: 20px (UI), 24px (buttons), 32px (headers)
- Color: Match text color or use Teal for CTAs
- Stroke weight: Default (consistent with Ionicons)

**Header:**
- Logo + "PoolDash" wordmark on left
- Teal background or white with teal accent border
- Navigation links in Slate 600

- Completed: Green background, white text

### Mobile (Technician App)

**App Icon:**
- Square logo (PoolDash icon only)
- Teal background with white icon mark
- 180px minimum size for app store

**Status Bar:**
- Dark (matches iOS/Android system)
- White status icons

**Navigation:**
- Bottom tab bar (iOS style)
- Teal accent for active tab
- Slate 400 for inactive tabs

**Cards:**
- White background, subtle shadow
- Border radius: 12px
- Padding: 16px
- 12px spacing between cards

### Marketing Materials

**Email:**
- Teal header with PoolDash logo
- Slate 600 body text
- Teal CTAs with "→" arrow
- Clean, minimal layout

**Social Media:**
- Square format: 1080x1080px (Instagram)
- Colors: Use full Teal palette
- Tone: Professional but friendly
- Include call-to-action

**Website:**
- Large hero image (pool/service related)
- Bold headings in Teal
- Clear CTAs throughout
- Mobile-responsive layout

## 8. Accessibility

### Color Contrast
- Text on background: Minimum 4.5:1 (WCAG AA)
- Large text: Minimum 3:1
- UI components: Minimum 3:1

### Contrast Examples
- ✅ Slate 900 on White: 18:1 (excellent)
- ✅ Slate 600 on White: 6.8:1 (good)
- ✅ Teal on White: 5.2:1 (good)
- ❌ Slate 400 on White: 3.6:1 (too low for body text)

### Testing
- Use WebAIM Contrast Checker
- Test with colorblind simulation tools
- Never rely on color alone for information

## 11. Dos and Don'ts

### Logo

| ✅ Do | ❌ Don't |
|------|---------|
| Use clear space around logo | Squish or stretch the logo |
| Scale proportionally | Rotate the logo |
| Place on appropriate backgrounds | Place on busy backgrounds |
| Use the official logo files | Recreate the logo from memory |

### Colors

| ✅ Do | ❌ Don't |
|------|---------|
| Use Teal for primary actions | Use multiple blues together |
| Ensure sufficient contrast | Rely on color alone |
| Use semantic colors correctly | Change color meanings |

### Typography

| ✅ Do | ❌ Don't |
|------|---------|
| Use Inter for all UI text | Mix multiple font families |
| Maintain consistent sizing | Use arbitrary font sizes |
| Use proper line-height | Let text be too cramped |
| Use 600+ weight for emphasis | Use italics for emphasis |

### Tone

| ✅ Do | ❌ Don't |
|------|---------|
| Be clear and direct | Use corporate jargon |
| Use active voice | Use passive voice |
| Focus on user benefits | Focus on features |
| Be friendly but professional | Be overly casual |
