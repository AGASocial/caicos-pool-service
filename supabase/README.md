# Supabase (Caicos monorepo)

All versioned SQL migrations for this project are in **`migrations/`** in this folder.

- Run the CLI from the **repository root** (`caicos/`), not from `admin-portal/`.
- `docs/schema.sql` is a full reference snapshot; incremental changes belong here as new timestamped files under `migrations/`.

Historical copies under `docs/migrations/` were consolidated here; see `docs/migrations/README.md`.
